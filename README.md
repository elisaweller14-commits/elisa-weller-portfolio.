# Elisa Weller — Portfolio personal

Portfolio web personal desarrollado como proyecto final de la asignatura Tecnología de Diseño para Internet.

El sitio presenta una selección de proyectos de diseño de interiores, identidad visual, diseño gráfico, packaging y diseño de producto. Cada proyecto incluye una página individual con contexto, proceso, resultado y galería de imágenes.

## URL en vivo

https://elisaweller14-commits.github.io/elisa-weller-portfolio./

## Proyectos incluidos

- El Refugio
- La Casa Encendida
- Let There Be Light!
- OCTOPUS Jeans
- Torres Blancas
- Coffee Party @ Ambu
- Vintage Packaging

## Tech stack

- HTML5 semántico
- CSS3
- Flexbox
- CSS Grid
- JavaScript
- IntersectionObserver
- Git
- GitHub Pages

## Funcionalidades

- Galería de proyectos generada dinámicamente desde un array de objetos JavaScript.
- Filtrado por disciplina mediante `.filter()` y re-render completo de la galería.
- Páginas individuales para cada caso de estudio.
- Galerías responsive.
- Lightbox para ampliar las imágenes.
- Animaciones de entrada al hacer scroll con `IntersectionObserver`.
- Navegación responsive con menú hamburguesa.
- Formulario de contacto con validación.
- Favicon personalizado.
